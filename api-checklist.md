# Чек-лист API-тестування контактних форм

**Контактні форми:**

a. **«Депозитна консультація»**

Ендпоінт: /catalog/htmx/send_feedback/41

b. **«Заявка на консультацію»**

Ендпоінт: *не визначався задля уникнення виконання тестових запитів на продакшн-сервер*

##

**Валідація поля «ПІБ» / «Імʼя»**
* Позитивні:
    * Граничні значення довжини вводу, кирилиця: 
        * Форма «Депозитна консультація»:
            * 3 символи;
            * *Відсутнє обмеження максимальної кількості символів*;
        * Форма «Заявка на консультацію»:
            * 2 символи;
            * 40 символів;
    * Апостроф;
    * Дефіс;
    * Літери ґ, є, ї, і.
* Негативні:
    * Граничні значення довжини вводу, кирилиця: 
        * Форма «Депозитна консультація»:
            * 2 символи;
            * 20 млн символів;
        * форма «Заявка на консультацію»:
            * 1 символ;
            * 41 символ;
    * Латиниця;
    * Цифри;
    * Спеціальні символи;
    * Емодзі;
    * Введене значення складається лише з пробілів;
    * Введене значення містить зайві пробіли:
        * пробіли на початку та вкінці;
        * декілька пробілів поспіль між словами.

**Валідація поля «Номер телефону»**
* Позитивні:
    * Валідний номер (12 цифр у форматі 380XXXXXXXXX);
* Негативні:
    * Неповний номер (11 цифр);
    * Перевищення максимальної довжини (13 цифр);
    * Невалідний формат;
    * Кирилиця;
    * Латиниця;
    * Спеціальні символи;
    * Емодзі;
    * Введене значення складається лише з пробілів;
    * Номер починається та закінчується пробілом.

**Обов'язковість полів**
* Порожнє поле «ПІБ» / «Імʼя»;
* Порожнє поле «Номер телефону»;
* Порожні поля «ПІБ» / «Імʼя» та «Номер телефону»;
* Запит без згоди на обробку даних.

**Безпека**
* Передача HTML-тегів;
* Передача XSS-payloads;
* Передача SQL Injection-payloads.

**Типи даних**
* null замість значення полів «ПІБ» / «Імʼя» та «Номер телефону»;
* Невідповідний тип даних;

**Конфігурація запиту**
* Виклик ендпоінту форми невалідними методами;
* Відправка запиту з некоректним заголовком Content-Type.
